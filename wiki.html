About Me

Hey! Welcome to the Pyrelbird Software wiki. I’m Zfry, the person behind all this, and here’s a bit about what I do.

I run Pyrelbird Software (used to be called Team-Fire Software). I make tools and games, mostly in Python. My goal is to create things that are fun, customizable, and useful for other indie developers.

I am under heavy work currently working on alot at once so it will take longer to make the tools, games

Projects

  • Voxel-Box – A fun 3D voxel-based project I made. Easily my favorite project
  • MC2D – A 2D game inspired by sandbox worlds. Second favorite project
  • Team-Fire 3D – A 3D multiplayer deathmatch game

I also experiment with Python-based tools and try to make tools that make game development easier for everyone.

Vision

I want Pyrelbird Software to help indie developers make cool stuff without struggling with complicated tools. Everything I make is meant to be accessible, moddable, and fun to use.
